  ШАГ 1: ЗАПУСТИТЕ ПРИЛОЖЕНИЕ

Найдите в этой папке файл:

    📄 RUN_APP.bat

Дважды кликните по нему → приложение запустится!

При первом запуске:
- Создастся база данных
- Загрузятся примеры (столики, блюда, заказы)
- Откроется главное окно

═══════════════════════════════════════════════════════════════
  ЧТО ВЫ УВИДИТЕ
═══════════════════════════════════════════════════════════════

✨ Красивое окно с:
   - Заголовком "Restaurant Management System"
   - Левым меню (Tables, Reservations, Menu, Orders)
   - Таблицей со столиками
   - Кнопками для управления

🎮 Попробуйте:
   - Нажмите "Delete" на любом столике
   - Введите текст в поле "Search"
   - Переключитесь на другой модуль (Reservations, Menu, Orders)

═══════════════════════════════════════════════════════════════
  ШАГ 2: СОЗДАЙТЕ УСТАНОВЩИК (ОПЦИОНАЛЬНО)
═══════════════════════════════════════════════════════════════

Если хотите создать установщик для распространения:

1. Прочитайте файл:
   📄 КАК_СОЗДАТЬ_УСТАНОВЩИК.txt

2. Кратко:
   - Установите Inno Setup
   - Запустите PowerShell от администратора
   - Выполните: .\scripts\create-installer.ps1
   - Получите готовый .exe в папке "installer"

═══════════════════════════════════════════════════════════════
  ВОЗМОЖНОСТИ ПРИЛОЖЕНИЯ
═══════════════════════════════════════════════════════════════

✅ TABLES (Столики)
   - Просмотр всех столиков
   - Добавление/редактирование/удаление
   - Фильтр по расположению
   - Поиск по названию
   - Цветные индикаторы статуса

✅ RESERVATIONS (Резервации)
   - Просмотр резерваций
   - Создание новых резерваций
   - Отмена резерваций
   - Фильтр по дате
   - Информация о клиенте

✅ MENU (Меню)
   - Каталог блюд
   - Категории блюд
   - Цены и время приготовления
   - Фильтр по категориям
   - Поиск блюд

✅ ORDERS (Заказы)
   - Активные заказы
   - Создание заказов
   - Закрытие заказов
   - Фильтр по статусу
   - Общая сумма

✅ DASHBOARD (Панель)
   - KPI карточки
   - Статистика
   - Последняя активность

Для запуска из исходников:
  - Windows 10/11
  - .NET 8 SDK

  - Windows 10/11
  - ~200 MB свободного места
  - Ничего больше не нужно!

❓ Где хранятся данные?
   → В файле restaurant.db в папке приложения

❓ Как сделать резервную копию?
   → Скопируйте файл restaurant.db в безопасное место

❓ Можно ли удалить тестовые данные?
   → Да! Используйте кнопку Delete

❓ Приложение не запускается?
   → Убедитесь, что установлен .NET 8 SDK
   → Попробуйте от имени администратора

   → См. файл КАК_СОЗДАТЬ_УСТАНОВЩИК.txt
